직장인
직장인 서류확인 테이블입니다.
필수서류 1) 분양 계약서
2) 계약금 납부 영수증
3) 신분증 또는 운전 면허증
4) 분양 계약서상 도장
5) 주민등록등본 1통
6) 주민등록초본 1통 ( 전체 주소 변경,   인적사항 변경 사항 포함 )
7) 가족 관계 증명서 1통 ( 미혼 및 배우자 분리 세대인 경우 )
8) 건강 보험공단 홈페이지 또는 1577-1000번 ) -> 계약자 본인명의 발급
소득서류  1) 재직 증명서
 2) 세무서 발급 소득 금액 증명원,전년도 근로소득 원천징수 영수증( 회사 발행 ), 금년 입사한 경우는급여 표시 된 소득세 원천징수 확인서  ( 회사 발행 )  - > 이중에서 1가지만 제출
