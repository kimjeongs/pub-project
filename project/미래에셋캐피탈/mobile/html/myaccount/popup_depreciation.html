  • 반납 시 교체 또는 수리비는 고객의 부담이며, 원상복구 후 감가합니다.
  • 반납 이전 고객이 자비로 교체 또는 수리비를 고객이 직접 부담한 경우 감가율만 반영합니다.
  • 도장불량이란 긁힌 것, 벗겨진 것, 휘어지거나 들어진 것, 색깔을 바꾼 것, 동전 크기 이상의 찍힘, 우그러짐, 부분 스프레이 칠, 부식 등 정상적인 차량 상태와 상이한 경우를 말합니다.
  • 감가액 = [출고 시점의 차량가격(VAT포함) X 항목별 감가율 총합계]+ 수리 실비 입니다.
  • 상기 감가와는 별도로 차량의 내/외관상 원상복구가 필요한 부위에 대해서는 실비 감가하기로 합니다.
