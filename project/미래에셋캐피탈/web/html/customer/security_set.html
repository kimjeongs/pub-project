보안프로그램 설치

안전한 사이트 이용을 위해 보안프로그램을 설치해 주세요.

보안프로그램 설치 안내

우선 각 프로그램의 ‘자세히보기’를 클릭하시어 기능을 확인하시기 바랍니다.
[전체설치]를 클릭하시면 자동으로 설치가 진행됩니다.
보안프로그램이 제대로 설치되지 않았을 경우, 삭제 후 다시 설치를 진행해 주세요.

통합 설치 프로그램

보안프로그램 통합 설치 프로그램 테이블입니다.
프로그램명 기능 설치현황
통합설치프로그램
(VeraPort)		 통합설치 프로그램은 미래에셋캐피탈에서 안전한 전자금융거래를 위해 제공하는
보안 프로그램들에 대해 쉽고 편리하게 설치할 수 있도록 하는 프로그램입니다.
관련문의처
위즈베라 고객센터 : 070-4659-3607(이용시간 : 평일 9시 ~ 18시 / 토, 일요일 휴무)
공인인증 프로그램
(SKCertService)		 공인인증 프로그램은 공인인증서 로그인과 거래내역에 대한 전자서명을 위한
프로그램 입니다.
관련문의처
코스콤 고객센터 : 1577-7337 (이용시간: 평일 8시~19시 / 토요일 9시~13시 / 일요일 휴무)
키보드보안, PC방화벽,
FDS(이상금융거래탐지)
(nProtect Online Security)
키보드보안
키보드로 입력되는 로그인 비밀번호, 공인인증 비밀번호 등을 암호화하여
중요정보가 유출되는 것을 방지하는 프로그램입니다.
PC방화벽
비인가된 접근을 차단하고 해킹툴 및 바이러스를 검색 및 치료해주는
프로그램 입니다.
FDS(이상금융거래탐지)
전자금융거래를 하는 PC의 정보를 수집하는 프로그램입니다.
관련문의처
잉카인터넷 고객센터 : 1566-0771 (이용시간: 평일 9시~18시 / 토,일요일 휴무)
설치완료
메모리 해킹방지 프로그램
(Ahnlab Safe Transaction)		 메모리 해킹방지 프로그램은 전자금융거래시 메모리에 저장된 계좌번호,
비밀번호 등의 중요정보를 해킹으로부터 보호하는 프로그램입니다.
관련문의처
안랩 고객센터 : 1577-9451 (이용시간: 평일 9시~18시 / 토,공휴일 휴무)

보안프로그램 선택 설치 프로그램 테이블입니다.
프로그램명 기능 설치현황
증명서위변조방지 프로그램
(TrustCertificater)		 증명서 위변조방지 프로그램은 고객님의 증명서 및 잔고내역을 출력시
위변조가 안되었음을 증명하는 보안프로그램 입니다.
관련문의처
에스지에이솔루션즈 고객센터 : 070-7308-9204~9208
