추가대출 상품 설명

한도내 추가대출이란?

  • 초기 대출 신청 시 고객님이 설정한 대출설정금액에서 대출금액 차액을 추가대출 받으실 수 있습니다.
    Ex) 대출설정금액 : 50,000,000원 / 대출금액 : 30,000,000원일 경우 : 한도내추가대출은 20,000,000원까지 신청가능

한도증액대출이란?

  • 초기 대출 신청 시 고객님의 대출한도금액에서 현재대출금액 차액만큼 추가대출 받으실 수 있습니다.
    Ex) 대출한도금액 : 100,000,000원 / 대출금액 : 50,000,000원일 경우 : 한도증액대출은 50,000,000원까지 신청가능 대출설정금액은 100,000,000원으로 변경
