    인증서 가져오기
  • 고객님의 PC에 저장되어 있는 공인인증서를 스마트폰으로 복사하면 스마트폰과 PC에서 동일한 공인인증서를 이용하여 대출신청을 받으실 수 있습니다.
  • 인증서 복사 진행 시 데이터 케이블 또는 이동식 외부장치 제거 후에 진행 부탁 드립니다.

인증서 가져오기 안내

  • 1미래에셋캐피탈 App에서 고객센터 > 공인인증서 관리 > 공인인증서 내보내기 선택

  • 2스마트기기에서 PC로 전송할 인증서 선택

  • 3“스마트기기/PC > PC인증서 가져오기” 버튼 실행

  • 4인증서 가져오기 팝업창의 인증번호 확인

  • 5PC에서 확인한 인증번호 입력

  • 6인증서 저장경로 선택 후 확인 버튼 실행
